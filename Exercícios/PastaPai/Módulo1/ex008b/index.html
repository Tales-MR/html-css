Código-fonte

Podemos colocar um código fonte escrito de uma forma melhor para leitura.
input() print() while() (Sem formatação)
input() print() While() (Com formatação)
Para isso utilizamos o <code>.
Podemos também fazer a mesclagem com a tag <pre> para que o HTML considere até mesmo as endentações contidas no texto, exemplo:

GameMaker Language

            
andar_esq = keyboard_check(ord(A));
if (andar_esq)
{
    x -= 1; 
};

Citações Simples/Completas

Podemos fazer o uso da tag <q> para demonstrarmos uma citação simples, curta e sem quebra de linha.
Exemplo:
O pai de um amigo me disse uma vez: O que é isso meu filho
E podemos também utilizar a tag <blockquote> para fazermos citações completas, que podem conter até mesmo o endereço da citação.
Exemplo:
Como dizia o doido do telhado:

Nem todos precisam de um telhado, as vezes só precisam de uma única telha para dormir.
