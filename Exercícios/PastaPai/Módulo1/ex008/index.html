Marcador de Texto

podemos também criar uma região marcada do texto usando o MARK.

Texto Deletado

Podemos marcar um texto como excluído para demonstrar que outrora já foi utilizado, mas não mais, usando <del>.

Texto Inserido

Podemos demontrar ao usuário que um texto foi inserido recentemente usando o <ins>.

Sobrescrito e Subscrito

Podemos usar a tag <sup> para que um texto seja sobrescrito, e a tag <sub> para que seja subscrito
