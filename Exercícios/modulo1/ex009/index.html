Listas Ordenadas

Para utilizarmos listas ordenadas, a tag usada é
<ol>
Como podemos ver no exemplo abaixo:

  1. Acordar
  2. Planejar meu dia
  3. Fazer meu café

Os elementos que compoem uma lista devem ficar entre as tags
<li> ... </li>

Modelos de Listas Ordenadas

As listas ordenadas também possuem alguns modelos de ordenação, como no exemplo abaixo:
  1. Utilizando o type="A"
  1. Utilizando o type="a"
  1. Utilizando o type="I"
  1. Utilizando o type="i"

Ordem de início

Para definir o ponto de início de sua lista, basta usar o parâmetro start, como no exemplo a seguir 

        
        <ol type="1" start="5">
        <li>Comprar café<li>
        <li>Fazer café<li>
        </ol>
O resultado do código acima seria:

  1. Comprar café
  2. Fazer café

Listas Não Ordenadas

As listas não ordenadas, como o próprio nome indica, não possui ordenação.
A tag utilizada para demarcar tal lista é <ul>
Exemplo de lista não ordenada:

  • Existem alguns
  • Modelos de demarcação
  • Para listas não ordenadas

Para utilizar diferentes modelos para listas não ordenadas, utilizamos o parâmetro type, como nas listas ordenadas.
Os valores que podem ser utilizados são:

  • disque
  • circle
  • square

Listas Mistas

Podemos também utilizar as listas mistas, que nada mais nada menos são que listas dentro de outras listas, como no exemplo abaixo:

  • Principal
  • Principal
    1. Secundária
    2. Secundária
      • Terciária
        1. Quaternária
      • Terciária
    3. Secundária
  • Principal

Listas de Definição

São listas utilizadas como uma espécie de dicionário, onde temos um termo e sua descrição logo abaixo, como no exemplo a seguir:

Amor
Algo que não pode ser explicado de forma simples.
Ódio
O mais puro sentimento

A tag que cria essa lista é <dl>, e os componetes são:

<dt>
termo a ser definido
<dd>
descrição do termo
